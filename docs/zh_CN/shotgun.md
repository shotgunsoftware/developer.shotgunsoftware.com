# Shotgun 开发

管理制作时需要执行大量工作以使所有运作部分保持同步。

使用我们的 Shotgun REST API 或 Python API 直接关联您的数据，使用事件触发器框架自动执行重复性任务，并使用动作菜单项直接从 Shotgun 的 Web 界面启动应用。

建议您使用 Shotgun API 开发自己的服务、应用程序、模块和组件，以便在本服务上运行或与本服务一起运行，供您自己和授权用户使用。
